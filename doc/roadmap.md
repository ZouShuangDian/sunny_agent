## 功能演进矩阵

> 以方向维度审视 v1.0 → v4.0 的能力成长路径，确保每条线逻辑连贯、逐步深化

### 业务目标演进

| 版本 | 业务目标 | 关键词 |
|------|---------|--------|
| v1.0 | 通用对话 + 项目管理 + 定时任务 + 客诉分析 | **场景落地** |
| v2.0 | 质量追溯 + 8D/PDCA + 质量知识库 + 数据分析 Agent | **质量深化 + 数据分析** |
| v3.0 | 质量全覆盖（SPC + IQC/IPQC/OQC + 合规审计）+ 覆盖 3 家子公司 | **行业标杆** |
| v4.0 | 流程可视化 + 供应链管理 + 报告自动生成 | **跨域扩展** |

### 方向一：智能执行力演进

| 版本 | 核心能力 | 演进关键词 |
|------|---------|-----------|
| v1.0 | 双层引擎（L1+L3）+ 护栏体系 + 短期记忆 + 多 LLM | **能推理** |
| v2.0 | 企业知识库 RAG + 企业码表 + 历史数据分析（试点） | **懂企业** |
| v3.0 | LangMemory 长期记忆 + 上下文智能分析 + 主动提示 + Skill 自动提取（试点） | **有记忆** |
| v4.0 | 用户画像学习 + 自适应进化 + Skill 自动提取（规模化） | **会进化** |

> **演进逻辑**：能执行 → 理解企业上下文 → 记住用户偏好 → 自主学习进化

### 方向二：企业级管控 / 智能数据演进

| 版本 | 核心能力 | 演进关键词 |
|------|---------|-----------|
| v1.0 | 鉴权 + SSO + ABAC + 审计 + 可观测性 + 部署 | **安全可控** |
| v2.0 | 桌面版 + 数据 Connector + MCP 协议 + 云端沙箱 | **连接一切** |
| v3.0 | 本体模型构建 + 本体驱动 RAG + 多模态数据集成（图片/文档/统一查询） | **语义理解** |
| v4.0 | 本体感知 Agent + 知识图谱可视化 + 过程数据仪表盘 | **知识图谱** |

> **演进逻辑**：安全基座 → 数据打通 → 语义理解 → 知识图谱可视化

### 方向三：开放扩展性演进

| 版本 | 核心能力 | 演进关键词 |
|------|---------|-----------|
| v1.0 | 工具注册 + Skill/Plugin/SubAgent 系统 + Skill Creator | **框架搭建** |
| v2.0 | Skills Hub + 自助 Skill 创建 + BP Plugin 共建（≥6 个） | **生态启动** |
| v3.0 | Hub 持续运营 + 覆盖度 50% + 质量体系 + 跨部门复用 | **生态繁荣** |
| v4.0 | 覆盖度 80% + 官方认证分级体系 | **生态成熟** |

> **演进逻辑**：搭框架 → 启动生态 → 运营繁荣 → 走向成熟

## Now — 正在做（2026 Q1）

> **信心度：高** | 范围和时间表已锁定

### v1.0 已完成项（21 项）

**业务目标**
- [x] 通用对话（L1/L3 双层执行）
- [x] 项目管理和文档对话（项目 CRUD + 文件管理 + SHA256 去重 + 数据隔离）
- [x] 定时工作任务（Cron CRUD + 执行历史 + 内置工具 + 会话关联）

**智能执行力**
- [x] IntentEngine 意图识别 + 路由决策（standard_l1 / deep_l3）
- [x] L1 FastTrack + L3 ReAct 双层推理引擎
- [x] 上下文压缩（双层漏斗：内存剪枝 + 摘要截断）
- [x] Todo 三层机制（宪法层 + 感知层 + 干预层）
- [x] 护栏层：JSON 修复 + Schema 校验 + 数值交叉校验 + 幻觉检测
- [x] 记忆系统：Redis 热存储 + PostgreSQL 冷存储 + L3 步骤持久化
- [x] 多 LLM 提供商支持（LiteLLM 统一接口：DeepSeek / OpenAI / Claude）

**企业级管控**
- [x] JWT 鉴权 + 角色权限控制 + SSO 单点登录（CAS 协议）
- [x] ABAC 数据隔离（按公司/事业部）+ 审计日志
- [x] Prometheus 可观测性（9 个指标 + structlog + trace_id）
- [x] Docker Compose 一键部署 + 健康检查

**开放扩展性**
- [x] 15 个内置工具 + ToolRegistry + tier 过滤
- [x] Skill 系统（DB 驱动 + SkillCallTool 运行时调用）
- [x] Plugin 系统（ZIP 上传 + 命令路由 + 管理 API）
- [x] SubAgent 系统（local_l3 / local_code / http 三种执行模式）
