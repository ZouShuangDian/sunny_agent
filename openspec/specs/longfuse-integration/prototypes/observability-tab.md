# 可观测性 Tab 原型设计

## 页面位置

管理面板 → 系统设置 → 可观测性

## 布局结构

### 1. Langfuse 状态卡片

```
┌─────────────────────────────────────────────────────────┐
│ Langfuse                                                │
│ Agent 执行链路追踪与监控平台                              │
│                                                         │
│ ● 运行正常                                               │
│ ↗ 打开 Langfuse 控制台                                   │
└─────────────────────────────────────────────────────────┘
```

- 状态指示器：绿色圆点 + "运行正常" / 红色圆点 + "服务异常"
- 跳转链接：点击在新标签页打开 Langfuse 控制台


### 3. 功能说明

| 元素 | 说明 |
|------|------|
| 时间范围 | 下拉选择：当天、7天、30天等 |
| 起始日期 | 日期选择器，默认当天 |
| 刷新按钮 | 手动刷新用量数据 |
| 总调用次数 | 选定时间范围内的 Agent 调用总数 |
| 总 Token 数 | 输入 Token + 输出 Token 汇总 |
| 预估费用 | 基于 Token 用量计算的成本预估 |
| 趋势图 | 按日维度的 Token 用量柱状图 |

### 4. 权限控制

- **管理员**：可查看所有用户的汇总数据，支持按用户筛选
- **普通用户**：仅能查看自己的用量数据
